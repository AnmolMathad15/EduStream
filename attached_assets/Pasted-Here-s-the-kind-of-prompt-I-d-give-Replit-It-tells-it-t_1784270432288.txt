# EduStream Dashboard UI Prompt (US21–US40)

You are continuing the existing EduStream LMS project.

**IMPORTANT: Do NOT redesign or replace the existing authentication pages.**

The login and signup pages are already complete with a premium dark SaaS aesthetic. Your task is to extend the exact same design language into the Student Dashboard.

---

# Design Goal

Build a world-class Student Dashboard that feels like it belongs to the existing EduStream authentication experience.

The user should feel like they have simply logged into the application—not switched to a different website.

The dashboard should resemble premium SaaS products such as:

* Linear
* Raycast
* Notion
* Clerk Dashboard
* Vercel Dashboard
* Stripe Dashboard
* GitHub
* Coursera Instructor Dashboard
* Udemy Learning Dashboard

Maintain the same spacing, shadows, gradients, animations, typography, rounded corners, color palette, and glassmorphism used throughout the authentication pages.

Never introduce a different visual language.

---

# Existing Theme (Must Keep)

Maintain:

* Dark charcoal background
* Glassmorphism cards
* Blue/Cyan gradients
* Rounded 16–24px corners
* Soft shadows
* Gradient buttons
* Smooth transitions
* Premium typography
* Existing color tokens
* Existing animations
* Existing ThemeProvider
* Existing reusable UI components

Reuse the current components whenever possible.

Do NOT create duplicate button, card, input, modal, or toast components.
